# spgsp
Doctrine 2 PostGIS simple support for spatial types and functions

##Install

Repository currently marked as dev. Set up your ```composer.json``` consequently

```javascript
"minimum-stability" : "dev",
"prefer-stable" : true,
 "repositories": [
    {
                "type": "vcs",
                "url": "https://github.com/pbald/spgsp"
    }
]
```
